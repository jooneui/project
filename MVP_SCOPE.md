# MVP Scope

## In scope

- landing page with product intro
- create portfolio flow without login
- single portfolio creation session
- URL submission form
- server-side metadata extraction
- editable card fields:
  - title
  - summary
  - tags
- delete card
- responsive card grid
- public share page with slug
- graceful fallback states for missing thumbnail / metadata

## Out of scope

- user accounts
- saved drafts across devices without share link
- multiple portfolios per user account
- collaboration
- AI tagging
- AI summarization as default behavior
- password-protected pages
- usage analytics UI
- monetization

## Stretch goals

- drag-and-drop ordering
- tag filtering
- import multiple URLs at once
- duplicate detection
- copy-to-clipboard share CTA
